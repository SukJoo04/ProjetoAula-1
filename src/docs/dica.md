1° Abrir o VSCode
2° No terminal digitar: cd desktop
3° No terminal dentro de desktop: mkdir porjeto
4° No terminal: cd projeto 
5° Dentro da pasta projeto: code .
6° No terminal da pasta projeto: touch index.html 
7° No terminal: mkdir src
8° No terminal: cd src
9° No terminal: mkdir css
10° No terminal: mkdir js
11° No terminal: cd css
12° No terminal: touch estilo.css
13° No terminal: cd..
14° No terminal: cd js
15° No terminal: touch script.js
16° No terminal: cd ..
17° No terminal: cd .. até chegar na raiz do projeto 

18° No terminal: git init
19° Se etiver na Fiap: git config --global user.name "Vinycius"
20° Se estiver na Fiap: git config --global usar.email vinyciuslu@gmail.com
21° No terminal: git status
22° Se estiver vermelho: git add .
23° Se estiver verder/azul: git commit -m "Mensagem do que você fez"
24° Entrar no GitHub em repositórios, criar um novo repositório, colocar o nome e deixar público, clicar no botão criar repositório (DICA DE OURO - NUNCA TRADUZA A PÁGINA DO GITHUB)
25° No bloco que estiver as 3 linhas:
    1° linha- Conecta sseu gitbash com seu GitHub
    2° linha- muda o nome master para main
    3° linha- pega tudo que estiver no projeto e sobe para o GitHub
26° Caso você esteja em outro pc: no seu repositorio no botão verde(code), copie o link 
27° Abra o seu VSCODE no terminal cd desktop: git clone (cole o link), isso vai clonar seu projeto e pode continuar de onde parou

    1º git status
